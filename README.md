Lasta Doc
=======================
The engine library for generating LastaDoc, called 'lasta-doc engine'

# Information
## Maven Dependency in pom.xml
```xml
<dependency>
    <groupId>org.lastaflute.doc</groupId>
    <artifactId>lasta-doc</artifactId>
    <version>0.3.7</version>
</dependency>
```
